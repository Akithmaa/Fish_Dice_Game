# Fish Dice Game

Fish Dice Game is a fun, simple dice game with an ocean theme. Roll the dice, earn points, and try to get the best score!

## Features
- User registration & login
- Session-based authentication
- Dice rolling gameplay
- Score calculation
- Simple and easy to play

## How to Play
1. Start the game
2. Roll the dice
3. The game calculates your score

## Tech Stack
- Backend - Node.js
- Frontend - HTML, CSS, JS
- database - Firebase
